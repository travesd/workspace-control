# Sanitized Workspace Config

This directory captures non-secret configuration that explains how the workspace is wired.

Files:

- `mcp.json` - workspace MCP server configuration for the detection browser wrapper.
- `codex.config.toml` - project-local Codex MCP configuration.
- `claude.settings.json` - project-local Claude settings. Currently empty.

Do not add provider auth files, API keys, browser tokens, DB env files, or Cloudflare Access credentials here.
