# Workspace Overlays

Workspace overlays hold the local rules, tools, skills, and profiles for one
concrete workspace.

Each workspace overlay should explain:

- what product or environment it governs,
- which core skills and specs it uses,
- which local tools and paths are expected,
- which provider adapters can consume it,
- what live activation target it maps to.

The current overlay is `detection-platform-metal/`.
