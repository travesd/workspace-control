# Pi Project Drafts

This `.pi/` directory is a draft project-local Pi configuration area.

It is intentionally minimal:

- `agents/` contains role prompts that mirror the Claude/Codex review pattern.
- `workflows/` contains example workflow JSON that can be adapted for Pi's workflow tool.
- `skills/` is reserved for Pi-specific skill mirrors if we decide not to load `agent-skills/skills/` directly.

Do not install or vendor third-party Pi packages here without review.
